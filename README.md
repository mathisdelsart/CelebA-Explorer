### 1. **Lab Exercises (TPs)**  
The TPs are designed to help you practice and apply the concepts taught in class. While some are not fully completed, they cover essential topics such as:  
- **Critiquing and Redesigning Visualizations** to improve clarity and communication.  
- **Clustering Techniques** to group similar data points based on their features.  
- **Non-Linear Dimensionality Reduction (NLDR)** to visualize high-dimensional data in lower-dimensional spaces.

### 2. **The Project**

#### **Project Theme**:
Develop an interactive data visualization app based on a real dataset. This year, we worked with the popular **CelebA dataset** to explore facial characteristics and implemented tools such as:  
- **Clustering images based on features**  
- **Similarity search between faces**  
- **Dynamic filtering and dimensionality reduction**  

This project demonstrates how raw data can be transformed into clear and actionable insights through visualization.
