## Project: Visualisation Tool for CelebA Dataset
This project is a visualisation tool for the CelebA dataset.

## Important Note

First, you need to make a folder named 'Datas' with the celebA dataset in it. You can download the dataset from [here](https://www.kaggle.com/jessicali9530/celeba-dataset). The name of this dataset must be rename to 'celeba_buffalo_l.csv'. Secondly, you also need to make a folder named 'assets' with all the images in it.

To run this project, you need first to install the prerequisites with the following command:
```bash
pip install -r requirements.txt
```
Then, you can run the project with the following command:
```bash
python3 app.py
```
